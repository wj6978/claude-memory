## 项目目标
智能机器人：硬件(STM32) + 视觉 + 听感 + 语音 + 本地LLM Agent

## 用户技术背景
- 51/STM32单片机基础
- 电脑配置：G1840 CPU + 2x Tesla M40 12G + 800元购入（性价比方案）
- 预算有限，追求性价比

## 系统架构

```
                    ┌─────────────────────────┐
                    │   Claude 大脑 (API)     │
                    │   支持视觉/语音/推理    │
                    └────────────┬────────────┘
                               │
                    ┌──────────┴──────────┐
                    │                     │
              ┌─────┴─────┐         ┌─────┴─────┐
              │  以太网    │         │  语音/图像 │
              │ (LLM通信) │         │   数据流   │
              └─────┬─────┘         └─────┬─────┘
                    │                     │
             ┌──────┴─────────────────────┴──────┐
             │           STM32F407 主控           │
             │  • 传感器采集 (MPU6050/AP3216C)    │
             │  • 运动控制 (GPIO/PWM)             │
             │  • 音频编解码 (WM8978 I2S)         │
             │  • 图像采集 (DCMI摄像头)           │
             │  • TF卡程序管理 (FatFS)            │
             └────────────────────────────────────┘
                          │           ▲
                   ┌──────┴───┐       │
                   │  串口指令 │       │ 指令下发
                   │   COM4   │       │
                   └──────────┘       │
                          ▲           │
                          │     ┌─────┴─────┐
                          └─────│   PC端    │
                                │ Claude CLI │
                                └───────────┘
```

## 重要发现

### SWD 烧录问题（已解决）
- **问题**：OpenOCD/pyOCD 通过 SWD 连接失败，芯片检测返回 PARTNO 0x0
- **解决**：使用 BOOT0 串口方式烧录正常工作

### DAP 仿真器烧录（已解决）
- **解决**：使用 **pyOCD** 烧录
- **命令**：`pyocd flash -t stm32f407igtx firmware.bin`
