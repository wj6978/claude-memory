# 测试记录文件夹

## 用途
- 存放测试结果和验证方法
- 记录实验数据和结论
- 便于复现和参考

## 目录结构
```
testing/
├── README.md          ← 本文件
└── results/           ← 测试结果文件
    ├── stm32/         ← STM32相关测试
    ├── led/           ← LED控制测试
    ├── audio/         ← 音频相关测试
    └── network/       ← 网络通信测试
```

## 文件命名规则
`日期_项目_测试内容.md`  
例如：`2026-04-24_stm32_dap_flash_test.md`
